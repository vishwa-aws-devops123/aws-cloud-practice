# Amazon S3 Basics

## What is S3?

Amazon S3 (Simple Storage Service) is AWS object storage.

## Uses

- File Storage
- Backup
- Static Website Hosting
- Log Storage

## Features

- High Durability
- Scalability
- Versioning

## Interview Answer

Amazon S3 is an object storage service used for storing files, backups, logs, and static website content.
