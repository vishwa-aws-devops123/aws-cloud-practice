# AWS Interview Questions

## What is EC2?

Amazon EC2 is a virtual server service provided by AWS.

## How do you connect to an EC2 instance?

Using SSH, a key pair, and the instance public IP address.

## What is IAM?

IAM is used to manage users, groups, roles, and permissions in AWS.

## What is S3?

Amazon S3 is an object storage service used for storing files, backups, logs, and static website content.

## What is a Security Group?

A Security Group acts like a firewall and controls incoming and outgoing traffic to AWS resources.

## Difference between Security Group and NACL?

* Security Group is stateful.
* Network ACL is stateless.
* Security Groups operate at instance level.
* NACLs operate at subnet level.

## What is a VPC?

A Virtual Private Cloud (VPC) is a logically isolated network within AWS.

## What is a Key Pair?

A Key Pair is used for secure authentication to EC2 instances.
